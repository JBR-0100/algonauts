Question of the Day

A. Floor Number

Problem Statement: Vasya wants to visit his classmate Petya, who lives in a large apartment building. Vasya knows the number of Petya's apartment, n. The building has only one entrance, and the apartments are distributed as follows:

  • The first floor contains exactly 2 apartments: apartments 1 and 2.
  • Each subsequent floor contains x apartments.

Apartment numbering starts at 1 on the first floor and continues sequentially upward.

Input:

The first line contains a single integer t (1 ≤ t ≤ 1000) — the number of test cases. Each of the following t lines contains two integers n and x (1 ≤ n, x ≤ 1000).

Output:

For each test case, output a single integer — the floor number of the apartment n.

Example:

Input Output
4
7 3
1 5
22 5
987 13 		3
1
5
77
